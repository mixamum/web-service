I have created a service that services a playlist for users to get certain songs and artists. The service is able to get certain information like the songs name, genre, artist from get request for the users to use. It also does operations like patch, delete and post certain songs. My service successfully updates and gets info from the mysql database. I believe my endpoints are names appropriately and supports all the CRUD operations for my service. I have created a Nginx servers that allows encrypted access and an endpoint that allows cross-origin custom middleware! I have successfully shared my github repository with twodee!

My Endpoints

  • POST /songs

    • Takes a JSON body as

    • "year": "2017",
    • "song_name": "Believer",
    • "artist_name": "Imagine Dragons"
    • "genre": "Rock",
    • "song_length": "3"

  • GET /songs
  • GET /:genre
  • GET /songs/:artist_name
  • GET /:genre/:song_name
  • GET /songs/:genre/:artist_name/
  • PATCH /songs/:id

    • Takes a JSON body as

    • "year": "2017",
    • "song_name": "Believer",
    • "artist_name": "Imagine Dragons"
    • "genre": "Rock",
    • "song_length": "3"

  • DELETE /songs/:id
